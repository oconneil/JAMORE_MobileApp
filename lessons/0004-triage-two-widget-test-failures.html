Failure A: Login ได้ false

test input ไม่มี CompanyID → FakeAuthSession.companyId == '' → AppState guard โยน AuthException → login คืน false

แก้ fixture ให้เป็น valid login โดยกรอก companyField และเพิ่ม validator ใน UI เพราะ backend DTO กำหนด CompanyID เป็น required

Failure B: ListTile assertion

Material ancestor → JamoreCard DecoratedBox สีขาว → ListTile

ListTile วาด ink ลง Material แต่ DecoratedBox สีขาวอยู่ทับ จึงมองไม่เห็น splash แก้ shared abstraction ที่ JamoreCard ให้ตัว card เป็น Material ที่มี shape/color แล้วใส่ Padding ภายใน

Feedback loop

  1. แก้ Login fixture แล้วรันเฉพาะ Login test
  2. แก้ JamoreCard แล้วรันเฉพาะ language test
  3. เมื่อสอง test ผ่าน ค่อยรัน suite ทั้งหมด
