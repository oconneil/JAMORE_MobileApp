Debug JamoreApiClient ตาม request จริง

เป้าหมาย: หยุด execution ก่อนส่ง request แล้วตอบให้ได้ว่า URL, token และ response มาจากไหน

สิ่งที่ต้องรู้ก่อน

JamoreApiClient extends ApiClient และไม่ได้ override request() ดังนั้น request ของ customer จะเข้า ApiClient.request() เหมือนกัน ต่างกันตรง callback ที่คืน customer base URL และ TokenJamore

ค่าที่ควรเห็นก่อนส่ง

this.runtimeType == JamoreApiClient
uri == https://CUSTOMER/api/User/GetUser/USERNAME
requestHeaders['Authorization'] == 'Bearer <TokenJamore>'

อย่าคัดลอก token เต็มลง log หรือ screenshot: ตรวจเพียงว่าไม่ว่าง หรือดู prefix สั้น ๆ ใน Variables panel

วิธีเดินทีละชั้น

  1. Run app ด้วย Debug mode แล้ว login
  2. เมื่อหยุดที่ app_state.dart:74 ใช้ Step over จน configure เสร็จ
  3. Resume ไป UserRepository แล้ว Step into ที่ _apiClient.get(...)
  4. ที่ ApiClient.request ตรวจ URL/header ก่อนบรรทัด _client.send
  5. Step over await แล้วตรวจ status/body หลัง response กลับ
