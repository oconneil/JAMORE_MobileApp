Universe API: Bearer + tenant header

Headerหน้าที่แหล่งค่า
Authorization: Bearer ...พิสูจน์ตัวตนกับ [Authorize]TokenUniverse
x-companyid: JCORPเลือก tenant ให้ EF Core global query filterCompanyID จาก login

Flutter request

Future<Object?> getCompany(String companyId) {
  final encoded = Uri.encodeComponent(companyId.trim());
  return _apiClient.get(
    'Company/Get/$encoded',
    headers: {'x-companyid': companyId.trim()},
  );
}

ApiClient เติม Authorization จาก accessTokenProvider ให้อัตโนมัติ จึงไม่ควรส่ง token ซ้ำจาก feature repository

ผลลัพธ์บน wire

x-companyid: JCORP
Accept: application/json
