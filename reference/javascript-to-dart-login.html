Flow จริงใน JAMORE

ปุ่ม Login
↓ _login(): Future<void>
↓ AppState.login(): Future<bool>
↓ AuthRepository.login(): Future<AuthSession>
↓ ApiClient.post(): Future<Object?>
↓ HTTP response
↑ decoded JSON → session → true → เปลี่ยนหน้า

กฎอ่าน function

1. วงชนิดคืนค่า เช่น Future<bool>

2. หาทุก await เพราะเป็นขอบเขตที่ control ออกไปทำงานชั้นถัดไป

3. หา return และ throw/catch เพื่อรู้เส้นทางสำเร็จและล้มเหลว

4. หาการแก้ state เช่น location = '/dashboard' และ notifyListeners()
